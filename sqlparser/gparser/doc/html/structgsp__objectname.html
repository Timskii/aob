Detailed Description

Oracle: [schema.]object[.part][ link : database[<.domain>[...n]][]

object is the name of the object. schema is the schema containing the object. part is a part of the object. This identifier lets you refer to a part of a schema object, such as a column or a partition of a table. Not all types of objects have parts. dblink applies only when you are using the Oracle Database distributed functionality.

postal_code is attribute SELECT c.cust_address.postal_code FROM customers c;
